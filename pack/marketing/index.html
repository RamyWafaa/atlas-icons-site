Distribution

One library, five formats.

Same icon names. Same look. Pick the one that fits your stack.

Webfont

Plain HTML, classes like at-arrow-down. Zero JS.

<link rel="stylesheet" href="https://unpkg.com/@vectoricons/atlas-icons/style.css">

React

Tree-shakeable JSX components.

npm i @vectoricons/atlas-icons-react

Vue

Single-file Vue 3 components.

npm i @vectoricons/atlas-icons-vue

Flutter

Dart icon font integration.

flutter pub add atlas_icons

React Native

Native font + JSX wrapper.

npm i @vectoricons/atlas-icons-react-native

Source SVGs

The whole library as a ZIP. Use anywhere.
