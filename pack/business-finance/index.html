Frequently asked

Questions, answered

What is Atlas Icons?

Atlas Icons is a free open-source icon library by GetIllustrations. 2,660 hand-crafted designs across 41 packs, each available in three stroke weights (thin, regular, bold) for a total of 7,980 icons. Distributed as a webfont, React, Vue, Flutter, and React Native packages, a Figma plugin, and source SVGs.

Is Atlas Icons free for commercial use?

Yes. Atlas Icons is MIT licensed. Use it in personal, commercial, and open-source projects with no attribution required. Modify, redistribute, and build on it freely.

How many icons does Atlas Icons have?

2,660 unique icon designs in three weights each (thin, regular, bold) for a total of 7,980 icons. Organized into 41 themed packs covering UI, communication, business, ecology, medical, hand gestures, weather, transportation, and more.

Does Atlas Icons have a Figma plugin?

Yes. The official Atlas Icons Figma plugin is free on the Figma Community. Search any icon and drag it straight onto your design canvas.

How do I install Atlas Icons in React?

Run npm install @vectoricons/atlas-icons-react, then import any icon as a JSX component:

import { ArrowDown } from '@vectoricons/atlas-icons-react';

<ArrowDown size={24} color="#000" />

Tree-shakeable, so only the icons you actually use end up in your bundle.

Can I customize the colors and stroke weight?

Yes. Click any icon to open the customization modal. Switch between thin, regular and bold stroke weights, pick any color, and adjust the size from 16px to 160px. The downloaded SVG and copied markup bake in your chosen color and size.

How do I download a single SVG icon?

Click any icon in the grid. In the modal, customize color, size, and stroke weight, then click Download SVG to save the file locally or Copy SVG to grab the markup directly.

What's the difference between Atlas Icons and GetIllustrations packs?

Atlas Icons is the free open-source MIT-licensed line — perfect for shipping in any project without licensing concerns. GetIllustrations sells premium hand-drawn illustrations, custom icon packs, and design-system-ready assets for brand-specific commercial work.

Where can I report issues or contribute?

The icon source library is at github.com/Vectopus/Atlas-icons-font — file an issue or open a pull request there. The website source for iconsatlas.com is at github.com/RamyWafaa/atlas-icons-site.
